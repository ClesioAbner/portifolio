# Portfólio de Eclesio Pembelane

Este portfólio apresenta meus projetos e habilidades como desenvolvedor de software, com foco em **desenvolvimento web moderno**. Ele inclui trabalhos realizados usando **Java, JavaScript, TypeScript, React**, integração de APIs e boas práticas de desenvolvimento, com exemplos de interfaces **responsivas e funcionais**.

O objetivo é demonstrar minha capacidade de criar soluções digitais **eficientes, seguras e com impacto real**, além de destacar meu interesse contínuo em **segurança informática e inovação tecnológica**.

## 🚀 Tecnologias Utilizadas

- **Frontend:** React, TypeScript, HTML5, CSS3, Tailwind CSS  
- **Backe
- **Extras:** Framer Motion para animações, design responsivo e temas claro/escuro
